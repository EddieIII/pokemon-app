# REPO
# 🎮 Pokédex Web Application

เว็บแอปพลิเคชันสำหรับค้นหาและดูข้อมูล Pokémon ทั้งหมด 1,351 ตัว พัฒนาด้วย Next.js, TypeScript และ Material UI

## ✨ Features
- 📋 แสดงรายการ Pokémon ทั้งหมด 1,351 ตัว พร้อมระบบ Pagination
- 🔍 ค้นหา Pokémon แบบ Real-time
- 📊 หน้า Detail แสดง Stats, ประเภท, วิวัฒนาการ และเสียง
- ⏳ Skeleton Loading ขณะรอข้อมูลจาก API
- 📱 Responsive Design รองรับทุกขนาดหน้าจอ

## 🛠️ Technologies
- **Next.js** — Framework หลัก
- **TypeScript** — ภาษาที่ใช้พัฒนา
- **Material UI (MUI)** — UI Component Library
- **PokeAPI** — แหล่งข้อมูล Pokémon

## 🚀 Installation
```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO.git
cd YOUR_REPO
npm install
npm run dev
```
เปิดเบราว์เซอร์ที่ http://localhost:3000
